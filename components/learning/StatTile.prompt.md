**StatTile / StatRow** — the completion scoreboard. Small gray label over a bold value; `StatRow` lays several out with hairline dividers.

```jsx
<StatRow items={[
  { label: "Total Time", value: "58 Min" },
  { label: "Avg Score", value: "87%" },
  { label: "Modules", value: "5/5" },
]} />
```

StatTile: `label`, `value`, `align`.
