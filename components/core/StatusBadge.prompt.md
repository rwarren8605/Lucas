**StatusBadge** — compact pill that labels the state of a module, answer or item. Tinted semantic surface + matching text color.

```jsx
<StatusBadge status="success">Done</StatusBadge>
<StatusBadge status="solidBrand">Start</StatusBadge>
<StatusBadge status="neutral" icon="lock">Locked</StatusBadge>
```

Tones: `success` `error` `warning` `brand` `neutral` `solid` `solidBrand`.
